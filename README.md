# Algorithm Study
## 코드 업로드하기
* `Add file` 클릭 
* github에 바로 코드 짤 거면 `Create new file`, 업로드 할 거면 `Upload files`
* 처음 폴더 생성할 때: `Create new file` => 파일 이름 작성하는 란에 `Ch#`(챕터 번호)적고 `/` 입력, 본인 이름으로 하위 폴더 생성 후 파일 업로드하기
  * github내에서는 빈 폴더만 따로 생성하지는 못한다고 합니다. shell로 업로드하는 게 간단할 거 같아요!
* 폴더 생성 후 파일 드래그하거나 직접 타이핑해서 코드 업로드
